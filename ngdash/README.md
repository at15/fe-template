# Angular Dashboard

## Dev

````bash
# call ng new empty to use latest @angular/cli to create app called empty
make new-empty
make new-ngzorro
````

The app name is used in following places

- `package.json` `name`
- `.angular-cli.json` `name`, will be changed to `angular-cli.json`
- `e2e/app.e2e-spec.ts`
- `README.md`
- `src/index.html`
