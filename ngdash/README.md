# Angular Dashboard

## Dev

````bash
# call ng new <project> to use latest @angular/cli and create dependencies
make new
````

The app name is used in following places

- `package.json` `name`
- `.angular-cli.json` `name`, will be changed to `angular-cli.json`
- `e2e/app.e2e-spec.ts`
- `README.md`
- `src/index.html`
